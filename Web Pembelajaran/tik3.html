Pengenalan CSS Eksternal

CSS Eksternal adalah cara menulis aturan CSS dalam file terpisah dengan file HTML. File CSS ini kemudian dihubungkan ke file HTML menggunakan tag <link> yang diletakkan di dalam bagian <head>. CSS eksternal memungkinkan Anda untuk mengelola gaya halaman web dengan lebih terorganisir dan dapat digunakan kembali di banyak halaman web.

1. Struktur CSS Eksternal

CSS eksternal ditulis dalam file dengan ekstensi .css dan dapat digunakan pada banyak halaman web. Anda cukup menautkan file CSS tersebut ke file HTML menggunakan tag <link>. 

                <!DOCTYPE html>
                <html>
                    <head>
                        <title>Contoh CSS Eksternal</title>
                        <link rel="stylesheet" href="style.css">
                    </head>
                    <body>
                        <h1>Selamat datang di CSS Eksternal</h1>
                        <p>CSS Eksternal memungkinkan kita untuk mengelola gaya halaman web secara terpisah.</p>
                    </body>
                </html>

2. File CSS Eksternal

Berikut adalah contoh file style.css yang digunakan untuk memberi gaya pada halaman HTML: 

                body {
                    background-color: lightblue;
                    font-family: 'Arial', sans-serif;
                }
                h1 {
                    color: darkgreen;
                    text-align: center;
                }
                p {
                    color: darkred;
                    font-size: 18px;
                }

3. Keuntungan Menggunakan CSS Eksternal

- Dapat digunakan pada banyak halaman: CSS eksternal memungkinkan Anda untuk mengelola gaya yang konsisten di seluruh halaman dalam sebuah website.

- Lebih terorganisir: Dengan memisahkan file HTML dan CSS, pengelolaan proyek web menjadi lebih terstruktur dan mudah dikelola.

- Lebih efisien: CSS eksternal memungkinkan browser untuk menyimpan file CSS, yang mempercepat waktu muat halaman jika file tersebut sudah di-cache.

4. Contoh Penggunaan CSS Eksternal

Berikut adalah contoh lebih lengkap tentang bagaimana menerapkan CSS eksternal pada halaman HTML:

                <!DOCTYPE html>
                <html>
                    <head>
                        <title>Contoh Gaya CSS Eksternal</title>
                        <link rel="stylesheet" href="style.css">
                    </head>
                    <body>
                        <h1>Selamat datang di CSS Eksternal</h1>
                        <p>CSS eksternal digunakan di sini untuk memberikan gaya pada halaman ini.</p>
                    </body>
                </html>
