Pengenalan CSS Internal

CSS Internal adalah cara menulis aturan CSS di dalam elemen <style> yang terletak di dalam bagian <head> sebuah dokumen HTML. CSS internal sangat berguna ketika Anda hanya ingin menerapkan gaya untuk satu halaman web tanpa mempengaruhi halaman lain.

1. Struktur CSS Internal

CSS internal ditulis di dalam tag <style> yang diletakkan di dalam bagian <head> pada dokumen HTML. Berikut adalah contoh struktur dasar HTML dengan CSS internal: 

                <!DOCTYPE html>
                <html>
                    <head>
                        <title>Contoh CSS Internal</title>
                        <style>
                            body {
                                background-color: #f4f4f4;
                                font-family: Arial, sans-serif;
                            }
                            h1 {
                                color: blue;
                                text-align: center;
                            }
                            p {
                                color: green;
                                font-size: 18px;
                            }
                        </style>
                    </head>
                    <body>
                        <h1>Selamat datang di CSS Internal</h1>
                        <p>CSS Internal digunakan untuk memberi gaya pada halaman ini.</p>
                    </body>
                </html>

2. Penjelasan Elemen

- <style>: Elemen ini digunakan untuk menulis aturan CSS internal di dalam bagian <head> halaman HTML.

- body: Elemen ini memilih seluruh halaman web dan memberikan latar belakang berwarna abu-abu muda serta font default Arial.

- h1: Elemen ini menentukan gaya untuk tag <h1>, yaitu memberi warna biru dan menyusun teks di tengah.

- p: Elemen ini memberikan gaya pada tag <p>, yaitu memberi warna hijau dan ukuran font 18px.

3. Keuntungan Menggunakan CSS Internal

- Sederhana untuk satu halaman: CSS internal sangat mudah diterapkan ketika Anda bekerja dengan satu halaman HTML saja.

- Tidak memerlukan file eksternal: Anda tidak perlu membuat file CSS terpisah, cukup menulisnya di dalam file HTML itu sendiri.

- Mudah dikelola untuk halaman tunggal: Ketika halaman web Anda tidak terlalu besar atau rumit, CSS internal cukup efektif untuk pengelolaan gaya.

4. Contoh Penggunaan CSS Internal

Berikut adalah contoh yang lebih lengkap tentang bagaimana menerapkan CSS internal pada halaman HTML:

                <!DOCTYPE html>
                <html>
                    <head>
                        <title>Contoh Gaya CSS</title>
                        <style>
                            body {
                                background-color: lightgray;
                                font-family: 'Verdana', sans-serif;
                            }
                            h1 {
                                color: darkblue;
                                text-align: center;
                            }
                            p {
                                color: darkgreen;
                                font-size: 20px;
                            }
                        </style>
                    </head>
                    <body>
                        <h1>Belajar CSS Internal</h1>
                        <p>Dengan menggunakan CSS internal, Anda bisa memberikan gaya yang menarik pada halaman web Anda.</p>
                    </body>
                </html>
