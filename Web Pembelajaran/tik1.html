Pengenalan HTML

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat dan menyusun halaman web. HTML menggunakan elemen-elemen yang disebut tag untuk membungkus konten dan memberi makna atau fungsi tertentu pada teks atau objek.

1. Struktur Dasar HTML

Dokumen HTML memiliki struktur dasar sebagai berikut:

                <!DOCTYPE html>
                <html>
                    <head>
                        <title>Judul Halaman</title>
                    </head>
                    <body>
                        <h1>Selamat Datang di HTML</h1>
                        <p>Ini adalah paragraf contoh.</p>
                    </body>
                </html>

2. Elemen HTML

Elemen HTML dibuka dengan tag pembuka dan diakhiri dengan tag penutup. Beberapa elemen umum yang sering digunakan di HTML adalah <h1> hingga <h6> untuk judul dengan ukuran yang berbeda, <p> untuk paragraf, <a> untuk membuat tautan ke halaman lain, dan <img> untuk menyisipkan gambar.

3. Atribut HTML

Atribut HTML memberikan informasi tambahan pada elemen HTML. Misalnya, href digunakan pada tag <a> untuk menentukan URL tautan, src pada tag <img> untuk menentukan sumber gambar, dan alt pada tag <img> untuk memberikan deskripsi gambar.

4. Contoh Penggunaan HTML

Berikut adalah contoh sederhana HTML yang menampilkan halaman dengan judul dan paragraf:

                <!DOCTYPE html>
                <html>
                    <head>
                        <title>Belajar HTML</title>
                    </head>
                    <body>
                        <h1>Halo, Dunia!</h1>
                        <p>Ini adalah contoh dasar HTML.</p>
                    </body>
                </html>
