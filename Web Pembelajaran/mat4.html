Teorema Faktor

Teorema Faktor menyatakan bahwa jika polinom P(x) memiliki akar di x = a (yaitu, P(a) = 0), maka (x - a) adalah faktor dari P(x).

Contoh Soal

Tentukan apakah (x - 2) merupakan faktor dari P(x) = x3 - 3x2 + 4x - 8.

Penyelesaian:

Substitusi x = 2 ke dalam P(x) untuk menemukan sisanya:
P(2) = (2)3 - 3(2)2 + 4(2) - 8
= 8 - 12 + 8 - 8
= 0
Karena P(2) = 0, maka (x - 2) adalah faktor dari P(x).

Teorema Vieta

Teorema Vieta memberikan hubungan antara koefisien polinom dan jumlah serta hasil kali akarnya.

1. Rumus Dasar untuk Polinom Kuadrat

Jika P(x) = ax2 + bx + c dengan akar x1 dan x2, maka:

1. Jumlah akar (x1 + x2) = -b/a

2. Hasil kali akar (x1 * x2) = c/a

2. Contoh Soal

Temukan jumlah dan hasil kali akar dari P(x) = 2x2 - 4x + 2.

Penyelesaian:

Koefisien a = 2, b = -4, dan c = 2.
Jumlah akar: (-(-4)/2) = 2
Hasil kali akar: (2/2) = 1

Latihan Soal

1. Tentukan apakah (x + 3) merupakan faktor dari P(x) = x2 + 3x - 18.

2. Temukan jumlah dan hasil kali akar dari polinom P(x) = 3x2 - 5x + 2.

Catatan: Gunakan latihan soal ini untuk memahami teorema faktor dan Vieta dengan lebih baik!
