Persamaan Garis Singgung

Persamaan garis singgung pada lingkaran adalah persamaan garis yang menyentuh lingkaran di tepat satu titik. Persamaan ini dapat ditemukan menggunakan beberapa metode tergantung pada informasi yang diketahui.

1. Rumus Dasar

Jika lingkaran memiliki persamaan (x - a)2 + (y - b)2 = r2, maka:

a) Garis singgung di titik (x1, y1) pada lingkaran:
(x - a)(x1 - a) + (y - b)(y1 - b) = r2

b) Jika persamaan garis singgung bergradien m:
Substitusi garis y = mx + c ke persamaan lingkaran, kemudian eliminasi untuk mendapatkan nilai c.

2. Contoh Soal

Tentukan persamaan garis singgung lingkaran (x - 2)2 + (y + 3)2 = 25 di titik (5, 1).

Penyelesaian:

Gunakan rumus:
(x - 2)(x1 - 2) + (y + 3)(y1 + 3) = r2
Substitusi: (x1, y1) = (5, 1), a = 2, b = -3, r = 5:
(x - 2)(5 - 2) + (y + 3)(1 + 3) = 25
3(x - 2) + 4(y + 3) = 25
3x - 6 + 4y + 12 = 25
3x + 4y + 6 = 0
Jadi, persamaan garis singgungnya adalah 3x + 4y + 6 = 0.

3. Latihan Soal

1. Tentukan persamaan garis singgung lingkaran x2 + y2 = 49 di titik (3, 4).

2. Sebuah garis singgung memiliki gradien -2 dan menyentuh lingkaran (x - 1)2 + (y - 2)2 = 10. Tentukan persamaan garis singgungnya.

3. Lingkaran (x + 1)2 + (y - 3)2 = 20 disentuh oleh garis singgung di titik (3, 1). Tentukan persamaan garis singgung tersebut.

Catatan: Kerjakan latihan soal ini untuk memahami konsep persamaan garis singgung dengan lebih baik!
